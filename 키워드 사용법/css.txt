margin : 개체의 외부 간격을 띄워줄 때 사용
 사용법
 1. 매개변수 1개 : 전면(상, 우, 하, 좌, 시계방향)의 간격 설정
 2. 매개변수 2개 : 가로(상, 하), 세로(좌, 우)의 간격 설정
 3. 매개변수 3개 : 상, 세로(좌, 우), 하의 간격 설정
 4. 매개변수 4개 : 전면 시계방향 순서대로 매칭 간격 설정
 5. margin-top, margin-right 등 세부 값을 일일히 설정

제외 선택자 : 태그, id, class, 선택자 등 선택 범주에서 제외범주를 제외하고 적용된다.
 사용법
 1. 선택자:not(제외태그,id,class 등){}
